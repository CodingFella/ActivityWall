## Getting Started

1. Prerequisites
 * Node.js (v18+)
 * PostgreSQL database

2. Installation
```bash
# Clone the repository
git clone https://github.com/CodingFella/ActivityWall.git

# Install dependencies
npm install
```

3. Environment Setup
Create a `.env` file in the root directory:

```
DB_USER=your_user
DB_PASSWORD=your_password
DB_HOST=localhost
DB_NAME=mapdb
DB_PORT=5432
```

4. Running the Project

```bash
# Start the backend server
node server.js

# Start the frontend
npm run dev
```
